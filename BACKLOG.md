## How to Use This File

**Humans:** Reorder the Ready list to change priorities. Move stories between sections as status changes. A merged PR means the story moves to Done.

**Agents:**
1. Read this file before starting work
2. Pick the top Ready story not already In Progress
3. Move it to In Progress with your name and branch
4. When PR is opened, leave it in In Progress
5. When merged by human, human moves to Done

**Dependencies:** A story can be started if all its dependencies are defined (Ready or beyond). They do not need to be implemented. Mock their interfaces.
